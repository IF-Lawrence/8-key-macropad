## How To Customize Key Icon

Each key in every Profile supports uploading a custom icon.

## When To Upload Icon To Device Manually

The configuration APP will automatically generate icons and copy them to the device.

If you prefer not to use the APP for configuration, or if you are using a Mac or Linux and configuring via the web, you may need to manually copy the icons to the deviceâ€™s USB drive.

## Naming rules for icon files

#### profile\_x\_key\_y.png

- x represents the Profile ID
- y represents the Key ID

For example:  
profile\_1\_key\_1.png means the icon for the first key in Profile 1.

## Where To Put Icon

All icon files are stored in the macropadâ€™s USB drive under the app\_icons directory.

After copying, you need to restart the keyboard for the changes to take effect.
